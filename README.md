# 3. :computer: 핵심 기능
* 음료를 생성할 수 있습니다.
    * 원하는 프랜차이즈의 음료 데이터를 넣을 수 있습니다.
* 프랜차이즈를 생성할 수 있습니다.
    * 원하는 프랜차이즈를 생성할 수 있습니다.
* 프랜차이즈, 음료 데이터 변경을 할 수 있습니다.
    * 프랜차이즈와 음료 데이터의 당, 칼로리, 사진 등을 수정할 수 있습니다.
